# Redes Neurais de Grafos

- Redes buscam aproveitar o grafo pra pegar uma embedding mais apropriada para o problema
- Uma GNN constrói uma rede neural baseada na topologia do grado de dados
	- Ou seja, os nós são conectados aos seus vizinhos de acordo com o grafo de dados.
- GNN tenta aproveitar melhor o conhecimento que está nos grafos de dados
- Combinam aprendizado baseado em grafos com aprendizado profundo

# Redes Neurais de Grafos Não Recursivas
- Numero fixo de camadas é aplicado sobre a entrada pra gerar a saída
	- Vantagem: não precisamos preocupar com convergência (já que o processo não é recursivo)
	- Desvantagem: adicionar muitas camadas pode resultar em um número elevado de parâmetros (pra resolver isso geralmente usam RN Convolucionais, ou CNNs)
- CNNs tem sido usado pra aprendizado de máquina voltadas para imagens (convoluções são usadas pra extrair características de uma região local)
- Convolução: operação mat. que envolve deslizar um pequeno conjunto de pesos (filto ou kernal) sobre uma entrada (como img) e calcular somas ponderadas locais. Serve pra destacar padrões locais, como bordas, texturas ou formas)
- CNNs são eficazes pra tarefas como:
	- deteção de objetos
	- segmentação de imagens
	- classificação de imgs;
	- reconhecimento facial;
- Funcionam bem pq conseguem detectar padrões locais.
- Funcionam por meio de um processo hierárquico de extração de característica
	- cada camada aprende a identificar padrões mais complexos da entrada
- Como CNNS podem ser usadas pra graph embedding?
	- CNNs são pra dados c/ estru em grade (como imgs)

# Mineração de Regras
- Descoberta de padrões significativos na forma de regras a partir de grandes coleções de conhecimento prévio
- Conjunto de arestas positivas e negativas -> dado
	- positivas: arestas observadas (dads ou implicadas no grafo de conhecimento)
	- negativas : definidas de acordo com uma determinada suposição de completude
- Regra segura:
	- é uma regra onde todas as var da cabeça (parte consequente; antecedente -> consequente; corpo -> cabeça) da regra aparece no copro positivo da regra.
	- Evita inferência "solta" com variáveis não instanciadas
