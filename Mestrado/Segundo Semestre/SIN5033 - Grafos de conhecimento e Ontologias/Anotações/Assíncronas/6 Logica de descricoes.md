# 1. Por que não usar LPO direto para ontologias?

A LPO é poderosa demais para esse fim:

Semi-decidível (o motor de prova pode não terminar), ela é verbosa para a modelagem, e também não é uma linguagem de marcação web (web markup language).

A solução foi pegar um **fragmento** da LPO que seja decidível e expressívo o suficiente
